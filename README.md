# 🧠 Skin Disease Prediction AI

This project is a Deep Learning based Skin Disease Detection system built using TensorFlow and Keras.  
The model is trained on skin disease images and can predict the disease from a given input image.  
The project demonstrates a complete machine learning workflow including preprocessing, augmentation, model training, evaluation, and prediction.

---

## 🚀 Project Highlights

- Developed deep learning classification model for skin disease detection
- Achieved ~92% accuracy on validation dataset
- Implemented image preprocessing and augmentation pipeline
- Optimized prediction pipeline for faster inference
- Stored trained model using .h5 format
- Designed modular ML workflow using separate scripts
- Visualized dataset distribution and performance metrics
